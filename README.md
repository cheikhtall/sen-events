SEN-EVENTS est une plateforme complète de gestion d’événements, conçue pour permettre à une entreprise spécialisée dans la location de matériel et la prestation de services événementiels d’optimiser ses opérations.
Le système permet d’administrer l’ensemble des ressources nécessaires à la réalisation d’événements : matériels, équipements, personnel, clients, réservations et facturations.

Le projet repose sur une architecture modulaire Spring Boot en backend.
